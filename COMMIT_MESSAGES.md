# Commit Message Template

Use this format for clear, meaningful commit messages:

```
[Topic/Difficulty]: [LeetCode #] [Problem Name]

Optional: Brief description of what was done
```

## Examples:

```
Tree/Medium: 938 Range Sum of BST

Add recursive DFS solution with BST optimization.
Time: O(h+k) | Space: O(h)
```

```
Arrays/Easy: 1 Two Sum

Add hash map solution and brute force approach.
```

```
DP/Hard: 123 Best Time to Buy and Sell Stock III

Add DP solution with state machine approach.
```

## Commit Command:

```bash
git commit -m "Tree/Medium: 938 Range Sum of BST"
```
